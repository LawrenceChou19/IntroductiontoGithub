Rearrange 
=================================

This module is used for rearranging names.

#examples

* Calling 'rearrange_name("Turing, Alan")' will return '"Alan Turing"'
* Calling 'rearrange_name("Hopper, Grace M.")' will return '"Grace M. Hopper"'
